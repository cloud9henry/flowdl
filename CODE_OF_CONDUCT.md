# Code of Conduct

## Our Pledge

We are committed to making participation in this project a harassment-free experience for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment:

- Being respectful and constructive in discussion
- Accepting feedback gracefully
- Focusing on what is best for the project and community

Examples of unacceptable behavior:

- Harassment, insults, or discriminatory language
- Publishing others' private information
- Trolling, intimidation, or personal attacks

## Enforcement Responsibilities

Project maintainers are responsible for clarifying and enforcing this code of conduct.

## Scope

This Code of Conduct applies within project spaces and in public spaces when an individual is representing the project.

## Enforcement

To report behavior that violates this code, contact the maintainers via GitHub issues or discussions with a private request.

Maintainers will review and investigate all complaints and respond appropriately.
