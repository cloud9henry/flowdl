# Security Policy

## Supported Versions

FlowDL is currently pre-1.0. Security fixes are applied to the latest commit on the default branch.

## Reporting a Vulnerability

Please do not report security issues publicly.

Report security vulnerabilities by opening a private security advisory on GitHub for this repository.

If private advisory reporting is unavailable, contact:

- cloud9henry@proton.me

Include:

- Affected version/commit
- Reproduction steps
- Impact assessment
- Suggested mitigation (if known)

We will acknowledge reports as soon as possible and provide updates during triage and remediation.
